# PRD (Product Requirements Document)
# My D-day

## 1. 제품 개요
- **제품명:** My D-day
- **목적:** 중요한 날짜들을 한눈에 관리하고 카운트다운을 확인하는 웹앱
- **타겟 사용자:** 본인 + 공유받은 지인

---

## 2. 사용자 스토리

| 번호 | 내용 |
|------|------|
| US-1 | 사용자는 로그인 후 내 D-day 목록을 볼 수 있다 |
| US-2 | 사용자는 D-day를 추가/수정/삭제할 수 있다 |
| US-3 | 사용자는 카테고리별로 D-day를 분류할 수 있다 |
| US-4 | D-day 당일에 푸시 알림을 받을 수 있다 |
| US-5 | 링크를 공유해 다른 사람도 내 D-day를 볼 수 있다 |
| US-6 | 공유 시 테마를 선택해 받는 사람에게 분위기를 전달할 수 있다 |

---

## 3. 기능 요구사항

### 필수 기능 (MVP)
- [ ] 회원가입 / 로그인
- [ ] D-day CRUD (생성, 조회, 수정, 삭제)
- [ ] 카테고리: 기념일, 시험, 사귀기 시작한 날, 생일
- [ ] D-day 목록 화면 (남은 날 / 지난 날 표시)
- [ ] 공유 링크 생성 + 테마 선택
- [ ] 공유 링크 수신자 전용 페이지 (로그인 없이 접근 가능)

### 추가 기능 (v2)
- [ ] 푸시 알림 (D-day 당일)
- [ ] 테마 커스터마이징 확장

---

## 4. 공유 기능 상세

### 공유 흐름
```
내가 D-day 선택
    ↓
테마 선택 (생일/기념일/시험 등)
    ↓
링크 생성 → 복사해서 카톡/문자로 전송
    ↓
받은 사람이 링크 클릭
    ↓
테마가 적용된 D-day 페이지 확인
```

### 공유 테마 목록

| 카테고리 | 테마 |
|----------|------|
| 생일 | 🎂 케이크, 풍선, 색종이 효과 |
| 기념일 | 💕 하트, 로맨틱한 분위기 |
| 사귀기 시작한 날 | 💑 달달한 느낌 |
| 시험 | 📚 깔끔하고 진지한 느낌 |

### 공유 페이지 화면 예시
```
┌─────────────────────┐
│   [테마 이미지/배경]   │
│                     │
│   🎂 철수의 생일      │
│                     │
│      D - 14         │
│   2025년 3월 20일    │
│                     │
│  "14일 후에 생일이에요"│
└─────────────────────┘
```
