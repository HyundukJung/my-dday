# PRD (Product Requirements Document)
# My D-day

> **상태:** MVP + 마일스톤 + PWA 배포 완료 (2026-04-15 기준)
> 운영 URL: https://my-dday.vercel.app

## 1. 제품 개요
- **제품명:** My D-day
- **목적:** 중요한 날짜들을 한눈에 관리하고 카운트다운을 확인하는 웹앱
- **타겟 사용자:** 본인 + 공유받은 지인

---

## 2. 사용자 스토리

| 번호 | 내용 | 상태 |
|------|------|------|
| US-1 | 사용자는 로그인 후 내 D-day 목록을 볼 수 있다 | ✅ |
| US-2 | 사용자는 D-day를 추가/수정/삭제할 수 있다 | ✅ |
| US-3 | 사용자는 카테고리별로 D-day를 분류/필터링할 수 있다 | ✅ |
| US-4 | 사용자는 시작일 기준 마일스톤(100/200/300/365일 등)을 자동 생성할 수 있다 | ✅ |
| US-5 | 링크를 공유해 다른 사람도 내 D-day를 볼 수 있다 | ✅ |
| US-6 | 공유 시 테마를 선택해 받는 사람에게 분위기를 전달할 수 있다 | ✅ |

| US-10 | 비밀번호를 잊으면 이메일로 재설정 링크를 받을 수 있다 | ✅ |
| US-11 | 로그인 후 비밀번호를 변경할 수 있다 | ✅ |
| US-12 | 각 D-day 카드에 간단한 메모를 남길 수 있다 | ✅ |

## 3. 기능 요구사항

### 필수 기능 (MVP) — 전부 배포 완료
- [x] 회원가입 / 로그인 (JWT)
- [x] D-day CRUD (생성, 조회, 수정, 삭제)
- [x] 카테고리: 생일 / 기념일 / 커플 / 시험
- [x] D-day 목록 화면 (남은 날 `D-n` / 지난 날 `D+n` / 당일 `D-DAY` 표시)
- [x] 공유 링크 생성 + 테마 선택 (4종)
- [x] 공유 링크 수신자 전용 페이지 (로그인 없이 접근)

### 확장 기능 (v1.5) — 배포 완료
- [x] **마일스톤 모드** — 시작일 + 100/200/300/365/500/1000일 자동 계산
  - 프리셋 체크박스 + 커스텀 일수 입력 (예: 777, 1234)
  - 카드에 "다음 마일스톤까지 N일" 표시 + 전체 목록 펼치기
- [x] **PWA** — iOS/Android 홈 화면 설치, 오프라인 캐시 (Service Worker)

### 확장 기능 (Phase 14) — 배포 완료
- [x] **비밀번호 찾기** — 이메일로 재설정 링크 (30분 유효, 1회용)
- [x] **비밀번호 변경** — 로그인 상태에서 현재 비밀번호 확인 후 변경
- [x] **카드 메모** — 각 D-day에 1000자 이내 메모 추가
- [x] **Google Calendar 추가** — OAuth 없이 캘린더 이벤트 생성 페이지로 이동 (all-day)

### 예정 기능 (v2)
- [ ] Web Push 알림 (D-day 당일, 마일스톤 도달일)
- [ ] Google Calendar **OAuth 자동 동기화** (현재는 링크 방식)
- [ ] 테마 커스터마이징 확장

## 4. 공유 기능 상세

### 공유 흐름
```
내가 D-day 선택
    ↓
테마 선택 (생일 / 기념일 / 커플 / 시험)
    ↓
링크 생성 → 자동으로 클립보드에 복사됨
    ↓
카톡·문자로 전송
    ↓
받은 사람이 링크 클릭 (로그인 불필요)
    ↓
테마가 적용된 D-day 페이지 확인
```

### 공유 테마 목록

| 카테고리 | 테마 이모지 | 분위기 |
|----------|------|------|
| 생일 | 🎂 | 케이크·풍선·색종이 효과 |
| 기념일 | 💕 | 하트·로맨틱 |
| 커플 | 💑 | 달달하고 따뜻한 톤 |
| 시험 | 📚 | 깔끔하고 진지한 톤 |

### 공유 페이지 예시
```
┌─────────────────────┐
│   [테마 이미지/배경]   │
│                     │
│   🎂 철수의 생일      │
│                     │
│      D - 14         │
│   2026년 5월 15일    │
│                     │
│  "14일 후에 생일이에요"│
└─────────────────────┘
```

## 5. 마일스톤 기능 상세

### 사용 예시
사귄 날을 `2026-02-17` 로 등록하면 다음이 자동 생성됩니다:

| 일수 | 계산 결과 |
|---|---|
| 100일 | 2026-05-28 |
| 200일 | 2026-09-05 |
| 300일 | 2026-12-14 |
| 365일 (1주년) | 2027-02-17 |
| 500일 | 2027-07-02 |
| 1000일 | 2028-11-13 |

### UI 특징
- 카드에 **D+85 (경과일)** + **다음: 100일 (D-15)** 표시
- `▼ 전체 보기` 클릭 시 모든 마일스톤 날짜 펼치기
- 지난 마일스톤은 취소선으로 구분

## 6. 비기능 요구사항
- **모바일 퍼스트** — 폰에서 우선 보기 좋게 디자인
- **미니멀 UI** — Pretendard 폰트, 파스텔 컬러
- **보안** — bcrypt 해시, JWT 인증, rate limiting, helmet 헤더
- **무료 호스팅** — Vercel + Railway 무료 플랜
- **PWA 설치 가능** — 홈 화면 아이콘, 전체화면 모드
- **오프라인 지원** — Service Worker로 정적 자원 캐시
